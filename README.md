# Dra. Cecília António - Plataforma de Agendamentos Jurídicos

Uma aplicação web full-stack desenvolvida para modernizar e automatizar a marcação de consultas jurídicas. Este projeto substitui processos manuais por um fluxo de agendamento digital self-service, integrado diretamente com a agenda da profissional.

🌐 **Live Demo:** [draceciliaantonio.com](https://draceciliaantonio.com/)

## 🎯 O Desafio
Profissionais da área jurídica perdem horas valiosas na gestão manual de marcações, trocas de emails e no alinhamento de disponibilidades. O objetivo deste projeto foi criar uma solução B2B/B2C que eliminasse este atrito, oferecendo uma interface limpa e intuitiva para os clientes finais, sem exigir que a advogada mudasse as suas ferramentas de trabalho habituais.

## 💡 A Solução e Funcionalidades
Desenvolvi uma plataforma à medida que atua como um assistente virtual 24/7:
- **Agendamento Self-Service:** Os utilizadores podem ver a disponibilidade em tempo real e marcar consultas de forma autónoma.
- **Sincronização Bidirecional:** Integração direta com a Google Calendar API. As marcações feitas no site aparecem instantaneamente na agenda da profissional.
- **Interface Responsiva:** Design otimizado tanto para dispositivos móveis como para desktop.
- **CI/CD Pipeline:** Deploy automatizado e contínuo configurado através da Vercel.

## 🛠️ Stack Tecnológica
- **Frontend & Framework:** Next.js, React, TypeScript
- **Integrações (API):** Google Calendar API (OAuth 2.0 / Service Accounts)
- **Deployment & Hosting:** Vercel
- **Controlo de Versões:** Git & GitHub
